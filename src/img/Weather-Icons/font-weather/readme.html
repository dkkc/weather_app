Weather Icons Font

How to install Weather Icons Font in your website

1. Copy the fonts and css folders from font-weather directory to your website project;

2. Include the CSS file reference in your header section of your site's HTML:

<link rel="stylesheet" href="css/font-weather.css">

3. Insert your desired icon using one of the following HTML code examples:

<i class="icon ic-sun"></i> <i class="icon" data-icon="&#xa001;"></i>

4. Style your icons: size, color, shadow and anything that's possible with CSS!
